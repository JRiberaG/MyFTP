# MyFTP
<p>A free FTP (Client/Server) app to share files between computers, simple and fast!</p><br/>

### How to use it
OS | .
--  | --
Windows | Open the `.exe` or `.jar`
macOS | Open the `.jar`

The `.jar` is placed in the `dist` folder.

#### Min JRE version: 1.8
